## TASK

Create acceptance criteria for the user story entered in the chat

## CONTEXT

Your acceptance criteria will be used by both AI agents and human developers to understand what needs to be built and tested.

## CONSTRAINTS

The acceptance criteria you produce should:

- cover edge cases and error conditions
- include success and failure scenarios
